# GRC & IAM Risk Oversight Portfolio

This repository demonstrates practical Governance, Risk, and Compliance (GRC) execution with a focus on Identity and Access Management (IAM) risk oversight, remediation tracking, audit readiness, and executive reporting.

## Scope
The portfolio showcases:
- Enterprise risk registers aligned with NIST CSF
- IAM-specific risk identification and access governance
- Remediation tracking and audit evidence preparation
- Executive-level risk and compliance reporting using Power BI

## Purpose
The goal of this portfolio is to demonstrate hands-on GRC execution and risk oversight workflows commonly used in enterprise environments, with an emphasis on clarity, traceability, and audit readiness rather than security engineering or system implementation.

## Data Disclaimer
All data in this repository is synthetic and created solely for demonstration and portfolio purposes. It does not represent any real organization, system, or customer data.
